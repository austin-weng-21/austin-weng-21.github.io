What are the differences between relative,
absolute, and fixed positioning?

Relative position

Relative position places the element relative to it's normal position in the document flow. Page scrolling does affect this position.

Absolute position

Absolute position places the element relative to it's closest parent element. In other words, it's container that encloses it inside. In order to use absolute position the parent element must be positioned with either relative or absolute. Absolute position will places the element relative to the browser viewport if it cannot find the positioned (relative/absolute) parent element. Page scrolling does affect this position.

Fixed position

Fixed position places the element at a fixed spot relative to the browser viewport. It removes the element from the normal document position completely.Page scrolling doesn't affect this position.
